Tools Used



  • Leaflet ~ Leaflet is an JavaScript library that allows developers to easily put interactive maps on their websites.
  • Nominatim API ~ Nominatim is an API that is used for looking up addresses or coordinates and providing information about hte location.
  • St. Paul Crime API ~ This is an API we developed that allows a client to look through a database of crimes that have occurred in the St. Paul area.
  • VueJS ~ Vue is a reactive JavaScript framework that is used to fill our table with data and apply filters in response to user input.

Interesting Findings



  1. Macalster-Groveland, the neighborhood housing our school, has the least amount of crime
  2. Majority of the crimes within this data are from proactive police visits
  3. Union Park has majority theft cases
  4. West Seventh neighborhood has more narcotic use and assault than other neighborhoods
  5. Payne-Phalen is the most violent neighborhood, home to majority of the highest crime classifications
  6. There is significantly more reported crime at 6pm than 6am, almost ten times the amount of crime compared to the morning
