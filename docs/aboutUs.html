Tools Used



  • Leaflet ~ Leaflet is an JavaScript library that allows developers to easily put interactive maps on their websites.
  • Nominatim API ~ Nominatim is an API that is used for looking up addresses or coordinates and providing information about hte location.
  • St. Paul Crime API ~ This is an API we developed that allows a client to look through a database of crimes that have occurred in the St. Paul area.
  • VueJS ~ Vue is a reactive JavaScript framework that is used to fill our table with data and apply filters in response to user input.
